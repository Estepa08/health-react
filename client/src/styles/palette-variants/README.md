# Варианты палитры (тестовые)

Каждый файл — полная копия `../tokens.css` с теми же именами переменных, но
перекрашенная под одну из обсуждённых пар цветов. Сайт по умолчанию их не
использует — `index.css` продолжает импортировать `./styles/tokens.css`.

## Как посмотреть вариант локально

В `client/src/index.css` временно замени импорт:

```css
@import './styles/tokens.css';
```

на нужный вариант, например:

```css
@import './styles/palette-variants/tokens.sun-stone.css';
```

и перезапусти `npm run dev`. Перед коммитом верни импорт обратно на
`./styles/tokens.css`.

## Файлы

| Файл | Пара цветов |
|---|---|
| `tokens.royal-blush.css` | Королевский синий `#3447AA` + пудровый розовый `#FBEAEB` |
| `tokens.sulu-pine.css` | Сулу `#9FE870` + Глубокая пихта `#163300` |
| `tokens.sun-stone.css` | Яркое Солнце `#FCDB32` + Большой Камень `#141D38` |
| `tokens.turquoise-black.css` | Бирюзовый `#34E0A1` + Чёрный `#000000` |
| `tokens.periwinkle-lavender.css` | Барвинок `#6260FF` + Лаванда `#E4E4FF` |
| `tokens.warm-terracotta.css` | Спокойная терракота: `#03363D` + `#BDD9D7` + тёплый акцент `#E67E52` |
| `tokens.cool-monochrome.css` | Спокойная синева: монохром `#03363D` → `#BDD9D7` + холодный акцент `#7FB5D6` |
| `tokens.vivid-contrast.css` | Живой контраст: `#03363D` + `#BDD9D7` + акценты для игр `#E8B93E` (успех) / `#C4574B` (ошибка) |

`--md-grad-success`, `--md-grad-warning` и `--md-confirm-green` оставлены
универсальными (зелёный/жёлтый как статусные цвета успеха/предупреждения) во
всех вариантах, кроме бирюзовой темы, где акцентный зелёный совпадает с
основным цветом палитры.
